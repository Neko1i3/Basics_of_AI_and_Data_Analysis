## IT Matura in Python

Matura is a nationwide exam in Poland, taken by every student at the end of highschool education. The IT Matura always consists of algorithmic, coding, spreadsheet and database tasks. Students need to know how to use different tools, usually Python/C++, Excel and Access.

I found it interesting that you only need Python to take on the whole exam (with the use of pandas, which is forbidden sadly). I don't think it's any easier or harder than using Microsoft Office, but to me, solving everything in Python seems much more elegant.

I kept everything in Polish, since the original statements are all in this language.
